docs/DESIGN.md의 디자인 토큰·타이포·컴포넌트·핵심 화면(Cohort Home)을 실제로 렌더링한 미리보기입니다. 브라우저에서 이 파일을 열어 확인하세요.

Color System

중립 기반 + 실행/신뢰의 녹청색. 보라/네온 그라디언트는 사용하지 않습니다.

Typography — Pretendard

대시보드 안에서는 hero 크기 제목을 쓰지 않습니다.

h3 / 205주차 강의 영상
body / 16다음 과제는 이 영상의 실습 내용을 기준으로 진행됩니다.
body-sm / 142026년 6월 14일 (토) 오후 2:00 · Zoom
caption / 12마감 임박 · 강사 검수 필요

Dialog
공개 범위를 변경할까요?
프로필이 전체 공개로 바뀝니다. 동문이 아닌 사용자도 회사명과 직책을 볼 수 있습니다.

Iconography — Lucide

제품 영역별 권장 아이콘. 기본 20px, 절제된 라인 스타일. 아이콘은 장식이 아니라 기능을 돕습니다.

Core Screen — Cohort Home

상태 → 이유 → 다음 행동 순서. 첫 화면 카드 5개 이하, primary CTA는 하나.
