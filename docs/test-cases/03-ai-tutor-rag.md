## ⚠ 알려진 갭

- **답변 저장·공유**: "대화방에 공유 / 코칭 예약 / 신고" 칩은 클릭해도 동작 없음 (D-20 미구현).
- **영상 Q&A 패널**(세션 상세 우측)은 별도 컴포넌트라 RAG 미연결 — 목업 응답.
- 대화 히스토리가 새로고침 시 소실 (클라이언트 state만, DB 저장 안 함).
- rag:sync는 수동 실행 — 파일 변동 자동 감지(스케줄러/watch)는 자동화 도구 과제.
